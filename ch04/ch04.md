# 对象 Object
- 面向对象 object-oriented
- object = attribute + method
  - `ball.kick()/throw()/inflate() ball.size/color/weight`
  - 属性是属于对象的变量，方法是属于对象的函数
  - 定义class，创建一个instance，这个实例就是一个对象
  - special method: __init__, __str__, __add__, __eq__
  - self
